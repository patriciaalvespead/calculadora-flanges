# Calculadora de Parafusos para Flanges

Ferramenta para dimensionamento de parafusos para flanges DIN EN 1092-1 e ANSI B16.5.

## Parâmetros de entrada
- Norma
- DN / NPS
- Classe de pressão
- Tipo de flange
- Tipo de junta

## Saídas
- Quantidade de furos
- Diâmetro dos furos
- Bitola do parafuso
- Comprimento mínimo recomendado

⚠ Sempre validar com desenho técnico e norma aplicável.
